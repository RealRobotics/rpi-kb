# Slideshow software for Raspberry Pi

Using an old Raspberry Pi to run slide shows at exhibitions or trade shows means that you don't have to leave your precious laptop lying around all day!  Here are a couple of solutions that I have tested.

The simplest and easiest to set up by far is PiSlideOS, <https://github.com/JarvyJ/pislide-os>.  This is ideal for using on a Pi Zero that is dedicated to displaying pictures, as it can do nothing else!  It should not be connected to the internet as it will become insecure.

The second solution is Video Looper, <https://term7.info/simple-videolooper-based-on-a-raspberry-pi-and-vlc/>.  This uses a standard Raspberry Pi OS image so can be connected to the internet and administered remotely.  The instructions are slightly out of date and do not always apply to the Pi that you are using.
